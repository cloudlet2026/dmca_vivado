此工程为双指数梯形成形算法的数字多道设计的能谱测量仪。硬件使用ALINX `AN9238`（AD9238 65MSPS） ADC模块（只使用通道1），搭配ALINX ZYNQ7020开发板使用。AN9238连接开发板`J21`扩展口。  
开发工具：Vivado2018.3。  
通讯方式：串口通讯，波特率115200。  
发送命令为：`dmca,[time],[ntr],[ntf],[na],[nb]`
`dmca,10,4,40,60,120`表示采集10秒，τr为4个采样点，τf为40个采样点，梯形成形上升时间长度为60个采样点，梯形成形平顶时间长度为120-60=60个采样点。  
返回数据为:`data:[0道]，[1道]，……[2047道]`  
（可自己根据需要修改道数）  

PS<-->PL之间AXI-Lite通信6个寄存器配置如下：  

| 寄存器 | 说明 |  
|------|------|
| `slv_reg0[0]`       | 是梯形成形算法的使能;                                 |
| `slv_reg0[4]`       | 是ADC采样模块的使能;                                  |
| `slv_reg0[8]`       | 是将能谱写入到BRAM的使能信号;                         |
| `slv_reg0[12]`      | 是PS接收到PL中断后，发出的清除中断标志;               |
| `slv_reg1[8:0]`     | 是na;                                                 |
| `slv_reg1[24:16]`   | 是nb;                                                 |
| `slv_reg2`          | 是 a/(na*(a-b)) （Q30）;                              |
| `slv_reg3`          | 是 1/(na*(a-b)) （Q30）;                              |
| `slv_reg4`          | 是 b （Q30）;                                         |
| `slv_reg5[13:0]`    | 是三角成形阈值，超过阈值是脉冲起始位置判断的条件之一; |
| `slv_reg5[23:16]`   | 梯子形平顶延时多长时间取平顶均值作为幅值;             |
| `slv_reg5[28:24]`   | 是三角成形连续超阈值计数;                             |

AD9238 ADC的输入范围是-5V\~+5V，数字输出范围是0\~4096，在adc_sample模块中将其转换为了（-2048\~+2048）14位有符号数。  
intr为PL接收到结束能谱写入后发出的中断，告诉PS端PL已经结束能谱生成，可读取BRAM中的能谱数据。PS触发中断后，读取能谱数据并清除中断intr（通过中断清除信号intr_clr），PS清除中断intr后发出中断响应intr_clr_resp将用于传输”中断清除“指令的寄存器清零，完成中断的控制。
